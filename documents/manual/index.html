Technology scores and maturity scores are typically generated on a subset of the available data, and often created for a set of domains (computers, molecular biology, mechanical engineering, and so on). In addition, scores are usually created for each year. What this amounts to is that we usually create a corpus of patents for a given year and a given domain and then process this corpus as a unit. The workflow outlined in the sections above follows that assumption.
